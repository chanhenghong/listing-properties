## Getting started Front-End Demo

### Installation

#### Clone project:

```bash
$ git clone git@bitbucket.org:z1data/recommendation-deployment-with-fastapi.git
```

#### Install [pnpm](https://pnpm.io/installation) via npm:

```bash
$ npm install -g pnpm
```

#### Install Package:

```bash
cd front-demo

$ pnpm install
```

#### Run for development:

```bash
$ pnpm dev
```

#### Run for deployment:

```bash
$ pnpm build
and

$ pnpm start
```
